# Корневая директория сайта

Здесь размещаются файлы вашего проекта Битрикс.

## Структура

После установки Битрикс здесь будут находиться:
- `bitrix/` - ядро Битрикс
- `local/` - ваши кастомные файлы и шаблоны
- `upload/` - загруженные файлы
- `index.php` - точка входа

## Разработка

Все файлы в этой директории автоматически доступны в контейнерах:
- В PHP контейнере: `/opt/www/`
- В Nginx контейнере: `/opt/www/`

Вы можете редактировать файлы напрямую на хосте - изменения сразу будут видны в контейнерах.
